SpaceJS
=======

This is a JS framework, define a new function to create application by js

Init
======

Require file:
- underscore.js
- require.js

Import file space.js to your index.html
Import app.js (your main file)
Using
=====

In app.js
<blockquote>
<p>namespace('APP').</p>
<p>  config({}).</p>
<p>  evi({}).</p>
<p>  import({}).</p>
<p>  defined(</p>
<p>    clazz('root').</p>
<p>    body({</p>
<p>        run: function(){</p>
<p>          alert('hello Space JS');</p>
<p>        }</p>
<p>    })</p>
<p>  ).</p>
<p>  run('root', 'run');</p>
</blockquote>
